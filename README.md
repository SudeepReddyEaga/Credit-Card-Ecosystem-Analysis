# Credit Card Ecosystem Analysis Dashboard

A comprehensive Power BI analytics project focused on analyzing customer spending behavior, transaction trends, revenue insights, and financial KPIs within a credit card ecosystem using interactive dashboards and business intelligence techniques.

## Features

* Interactive KPI dashboards
* Customer behavior analysis
* Transaction trend visualization
* Revenue and spending insights
* Dynamic filtering and drill-down analytics

## Technologies Used

* Power BI
* DAX
* Power Query
* Data Visualization
* Git & GitHub
